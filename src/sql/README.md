# BRICK BREAKER GAME - DATABASE

## Yêu cầu hệ thống
- SQL Server 2016 trở lên
- 100MB dung lượng ổ cứng

## Cách cài đặt
1. Mở SQL Server Management Studio (SSMS)
2. Chạy các file theo thứ tự:
   - 01_create_tables.sql
   - 02_sample_data.sql (nếu cần dữ liệu mẫu)
   - 03_stored_procedures.sql
   - 04_views_functions.sql
   - 05_triggers.sql
   - 06_indexes.sql

## Lưu ý
- File 02_sample_data.sql chỉ chạy 1 lần khi cần dữ liệu test
- Các file còn lại có thể chạy lại nhiều lần
- Nên backup database trước khi chạy
